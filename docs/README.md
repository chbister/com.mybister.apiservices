# GitHub Pages Starter für App-Infos

Dieses Paket enthält eine einfache statische Website für:

- Startseite mit Beispielinhalt
- Impressum
- Datenschutzerklärung
- 404-Seite

## Inhalte anpassen

Öffne und bearbeite diese Dateien:

- `index.html` → App-Beschreibung, Store-Links, Support-Mail
- `impressum.html` → deine echten Pflichtangaben
- `privacy.html` → deine tatsächliche Datenverarbeitung
- `assets/style.css` → Design/Farben/Abstände

## Veröffentlichung mit GitHub Pages

1. Neues GitHub-Repository anlegen, z. B. `app-info`
2. Alle Dateien aus diesem Ordner hochladen
3. In GitHub öffnen: `Settings` → `Pages`
4. Unter `Build and deployment` auswählen:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - `Folder`: `/ (root)`
5. Speichern
6. Nach kurzer Zeit ist die Seite erreichbar unter:
   - `https://DEIN-USERNAME.github.io/app-info/`

## Eigene Domain

Optional kannst du später eine eigene Domain verbinden.
Dann ergänzt du zusätzlich eine Datei namens `CNAME` im Projektstamm, z. B. mit folgendem Inhalt:

```text
app.deinedomain.de
```

## Wichtiger Hinweis

Die Dateien `impressum.html` und `privacy.html` sind Muster mit Platzhaltern und keine Rechtsberatung.
Bitte prüfe die Angaben vor Veröffentlichung.
